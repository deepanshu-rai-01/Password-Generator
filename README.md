# 🔐 Password Generator

A secure random password generator built using Python that creates strong passwords using letters, numbers, and special characters.

## 🚀 Features

- Generate secure random passwords
- User-defined password length
- Includes uppercase & lowercase letters
- Includes numbers
- Includes special characters
- Easy to use
- Input validation

## 🛠 Technologies Used

- Python
- random module
- string module

## ▶️ How to Run

1. Download the project.
2. Open Terminal or Command Prompt.
3. Run:

```bash
python PasswordGenerator.py
```
